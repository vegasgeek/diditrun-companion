A companion plugin for Did It Run? monitoring service.

== Description ==

This plugin is a companion to the Did It Run? monitoring service. It allows for an easy way to send check-in calls to the service.

== Installation ==

Upload the Did It Run? Companion plugin to your WordPress site, then activate it.

You're done!
